## 前置条件

- 注册 [beeos.ai](https://beeos.ai) BeeOS 账号
- 在 **设置 > API Keys** 中获取 **用户 API Key**（`oag_...`）

## 1. 安装 SDK

<CodeGroup>

```bash npm
npm install @beeos-ai/sdk
```

```bash go
go get github.com/beeos-ai/sdk-go
```

```bash curl
# 无需安装，直接使用 curl
```

</CodeGroup>

## 2. 列出可用 Provider

<CodeGroup>

```typescript TypeScript
import { BeeOS } from "@beeos-ai/sdk";

const client = new BeeOS({ apiKey: "oag_YOUR_KEY" });
const providers = await client.providers.list();

for (const p of providers) {
  console.log(p.meta.id, p.meta.name);
}
```

```go Go
package main

import (
    "context"
    "fmt"
    beeos "github.com/beeos-ai/sdk-go"
)

func main() {
    client := beeos.New(beeos.WithAPIKey("oag_YOUR_KEY"))
    providers, _ := client.Providers.List(context.Background())
    for _, p := range providers {
        fmt.Println(p.Meta.ID, p.Meta.Name)
    }
}
```

```bash curl
curl -s https://openapi.beeos.ai/api/v1/providers \
  -H "Authorization: Bearer oag_YOUR_KEY" | jq '.data'
```

</CodeGroup>

## 3. 部署智能体实例

<CodeGroup>

```typescript TypeScript
const instance = await client.instances.deploy({
  name: "my-first-agent",
  agentFramework: "beeos-claw",
  modelPrimary: "gpt-4o",
});

console.log("实例 ID:", instance.id);
console.log("状态:", instance.status);
```

```go Go
inst, _ := client.Instances.Deploy(context.Background(), beeos.DeployInstanceRequest{
    Name:           "my-first-agent",
    AgentFramework: "beeos-claw",
    ModelPrimary:   "gpt-4o",
})
fmt.Println("实例 ID:", inst.ID)
```

```bash curl
curl -s -X POST https://openapi.beeos.ai/api/v1/instances \
  -H "Authorization: Bearer oag_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-first-agent",
    "agentFramework": "beeos-claw",
    "modelPrimary": "gpt-4o"
  }' | jq '.data'
```

</CodeGroup>

## 4. 调用智能体

实例状态变为 `running` 后，即可向其发送消息：

<CodeGroup>

```typescript TypeScript
const result = await client.agents.invoke(instance.identityId, {
  message: "你好！你能做什么？",
});

console.log("回复:", result.text);
console.log("上下文:", result.context_id);
```

```go Go
reply, _ := client.Agents.Invoke(context.Background(), agentID, beeos.InvokeAgentRequest{
    Message: "你好！你能做什么？",
})
fmt.Println("回复:", reply.Text)
```

```bash curl
curl -s -X POST "https://openapi.beeos.ai/api/v1/agents/${AGENT_ID}/invoke" \
  -H "Authorization: Bearer oag_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "你好！你能做什么？"
  }' | jq '.data'
```

</CodeGroup>

获取流式响应，请设置 `Accept: text/event-stream`：

```bash
curl -N -X POST "https://openapi.beeos.ai/api/v1/agents/${AGENT_ID}/invoke" \
  -H "Authorization: Bearer oag_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -H "Accept: text/event-stream" \
  -d '{"message": "介绍一下你自己"}'
```

## 5. 清理资源

```bash
curl -s -X DELETE "https://openapi.beeos.ai/api/v1/instances/${INSTANCE_ID}" \
  -H "Authorization: Bearer oag_YOUR_KEY" | jq
```
